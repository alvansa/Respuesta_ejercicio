## 📋 Requisitos Previos
- Python 3.13+
- pip

## Instalación

### Clonar repositorio
```bash
git clone https://github.com/alvansa/Respuesta_ejercicio.git
cd mi-proyecto
```

## Linux/Mac
python -m venv venv
source venv/bin/activate

## Windows
python -m venv venv
venv\Scripts\activate


## Instalar dependencias
pip install -r requirements.txt
